Question-1: What are the different ways to select an element in the DOM?

1. getElementById()
2. getElementsByClassName()
3. querySelector()
4. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. We can get the content inside any tag on an HTML page using innerHTML.
2. We can get the text inside any tag on an HTML page using innerText.
3. We can get the text content inside any tag on an HTML page using textContent. That means, if there are any tags within the element, they won't be included, only the plain text will be retrieved.

Question-3: What is event delegation in the DOM?

An event listener is added to the parent element, and it can propagate to the child elements as well.

Question-4: What is event bubbling in the DOM?

An event listener is added to the child element, and it propagates step by step through the parent elements until it reaches the document.

uestion-5: How do you create, add, and remove elements using JavaScript?

You can create any element using document.createElement() and add it to the HTML page using appendChild(). Additionally, you can select any element using a selector and remove it using the remove() method.
